POLÍTICA DE PRIVACIDADE

Esta Política de Privacidade é fornecida pela Marinete's Clean House para cumprir os requisitos da Política de Dados do Usuário do Google Play. Ao usar nosso aplicativo, você concorda com os termos descritos abaixo.

1. Dados Coletados e Finalidades:

Coletamos os seguintes dados pessoais, fornecidos por você ou obtidos durante o uso do aplicativo:

Dados Pessoais Fornecidos Expressamente por Você:

  • Nome Completo
  • CPF
  • RG
  • Endereço com CEP
  • Telefone com DDD

Coletamos esses dados quando você realiza o pré-cadastro no site. Essas informações são usadas para usufruir das funcionalidades do aplicativo, bem como para fins de validação e segurança dos usuários, incluindo a conferência de antecedentes criminais.

Dados Pessoais Obtidos de Outras Formas:

  • Dados de Geolocalização
  • Memória dos atos praticados dentro do aplicativo

Esses dados são coletados quando você realiza a contratação através do aplicativo e são utilizados para garantir transparência e segurança no uso do aplicativo.

3. Dados Sensíveis:

Não coletamos dados sensíveis, como origem racial ou étnica, convicção religiosa, opinião política, filiação a sindicato, dados referentes à saúde ou à vida sexual, dado genético ou biométrico.

4. Coleta de Dados não Previstos Expressamente:

Eventualmente, outros tipos de dados não previstos nesta Política de Privacidade podem ser coletados com seu consentimento ou com base em outra base legal. Qualquer coleta adicional será informada aos usuários.

4.a Compartilhamento de Dados Pessoais com Terceiros:

Não compartilhamos seus dados pessoais com terceiros, exceto quando necessário para cumprir obrigações legais ou regulatórias.

4.b Por Quanto Tempo Seus Dados Serão Armazenados:

Armazenamos seus dados pelo tempo necessário para atingir as finalidades elencadas nesta Política, considerando os direitos dos titulares e as leis aplicáveis.

5. Bases Legais para o Tratamento de Dados Pessoais:

Todas as operações de tratamento de dados têm uma base legal conforme a LGPD.

5. Direitos do Usuário:

Você tem os seguintes direitos de acordo com a LGPD:

  • Confirmação da existência de tratamento
  • Acesso aos dados
  • Correção de dados incompletos, inexatos ou desatualizados
  • Anonimização, bloqueio ou eliminação de dados desnecessários, excessivos ou tratados em desconformidade com a lei
  • Portabilidade dos dados
  • Eliminação dos dados pessoais tratados com consentimento, exceto nos casos previstos em lei
  • Informação sobre o compartilhamento de dados com terceiros
  • Informação sobre a possibilidade de não fornecer consentimento e as consequências da negativa
  • Revogação do consentimento

1. Como o Titular Pode Exercer Seus Direitos:

Para exercer seus direitos, entre em contato conosco pelo seguinte meio:

  • E-mail: sac@marinetes.com.br

2. Medidas de Segurança no Tratamento de Dados Pessoais:

Empregamos medidas técnicas e organizacionais para proteger seus dados contra acessos não autorizados e situações de perda, destruição, extravio ou alteração. Possuímos certificação ISO 27001 e seguimos os mais elevados padrões de segurança da informação.

7. Reclamação a uma Autoridade de Controle:

Você pode apresentar reclamação à Autoridade Nacional de Proteção de Dados se se sentir lesado.

8. Alterações nesta Política:

Esta Política de Privacidade foi atualizada pela última vez em 08/08/2022. Reservamo-nos o direito de modificar estas normas, e você será notificado de qualquer mudança.

Agradecemos por utilizar o aplicativo Marinete's Clean House. Se tiver dúvidas ou preocupações sobre sua privacidade, entre em contato conosco.
